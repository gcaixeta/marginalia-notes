# O que é?

  Cobra é um poderoso construtor de CLI que ajuda na definição de comandos, argumentos, flags, auto-complete, etc. 

## auto-complete

  Para utilizarmos o auto-complete é necessário utilizar ou ValidArgs ou ValidArgsFunction. O ValidArgs lida com valores já declarados em compilação, trazendo mais velocidade na resposta. Já o ValidArgsFunction, faz uma mágica, permitindo que rodemos uma funcao para trazer a sugestão de auto-complete. Quando apertamos tab o shell chama o programa com um argumento "__complete" e o Cobra roda a função ValidArgsFunction relacionada com o Command.

## version

  Caso não adicionemos uma flag --version, o cobra adicionará um comportamento padrão que mostra a string contida na propriedade cmd.Version.

## annotations

  A propriedade cmd.Annotations é um mapa de string para string que funciona como um espaço disponível para carregarmos qualquer tipo de informação junto com a struct Command. Desta forma, se quisermos passar uma autenticação, podemos adicionar o dado em cmd.Annotations["autenticacao"] e acessarmos mais para frente no fluxo, sendo uma forma mais elegante de extensão do framework. 

## flags and persistent flags

  As flags servem somente para o command onde foram declaradas, enquanto a pFlag é persistente e herdada pelos comandos filhos.
