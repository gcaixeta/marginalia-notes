## Estudos sobre funções

### filepath.Dir(path string) string

  Retorna o diretório a partir de um path. Ou seja, se tivermos posse de um path referente a um arquivo, podemos obter o seu diretório a partir deste método, passando o path do arquivo por parâmetro.

### Scanner x Reader

  O scanner tem uma abordagem mais simples e prática, mas perde em controle. Normalmente, pode ler uma linha ou uma palavra por vez, tendo também um limite de 64kb por leitura. 
  Já o Reader, possui mais ferramentas para ler diversos tipos de input, lendo até mesmo bytes.
