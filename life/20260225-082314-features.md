### Define the today task

 - Once in the day, Life requests a yesterday review. In that moment it will prompt you each of the yesterday's events, giving a rate and, maybe, adding some comments on how it went.

 - In the morning, the life app asks you for the day tasks. Then we can populate de to-do list for the day and, whenever you end a task, it can offer you another one from the list. Whenever you add a task, it asks for the priority. That way, it offers you the most urgent ones first, following the idea that we should take the hardest tasks first in the morning.
