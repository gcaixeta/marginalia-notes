# DoGet

Try block {
    logga em nivel INFO "doGet - start"
    captura o parametro *fileName* do request como string
    captura o parametro *printable* do request e o transforma em boolean
    
    caso o *fileName* seja vazio, loga erro e lanca excecao *IllegalArgumentException*
    
    caso o *printable* seja NULL, loga erro e lanca excecao *IllegalArgumentException*

    loga que esta comecando o retorno do arquivo

    cria uma nova *File* passando o nome do arquivo
    caso o arquivo nao exista, loga o erro "file does not exists" e lanca excecao *FileNotFoundException*

    verifica se e *printable*. caso seja, verifica se o arquivo e pdf e, caso seja, cria um novo *PdfReader* e um *PdfStamper*. Configura o *stamper* 
    
    fecha o *stamper* e o *reader*

    se nao for printable, loga WARN "nao e possivel imprimir o arquivo. Esta funcao so funciona com PDF"

    se o logger estiver em modo debug, loga DEBUG "set header and content parameters"

    seta os headers do Response para enviar a resposta
    seta o mimetype a partir da extensao do arquivo
    faz o stream dos bytes do arquivo para o OutputStream do response e faz o flush() depois o close() do *OutputStream*

    loga DEBUG e tenta excluir o arquivo

    trata lancamento de erro SocketException ou Exception generico

# DoPost

    Chama metodo *doGet* passando request e response

    O **Util.gs** faz parte do pacote it.unipol.sx.gcs.mediaenrihment.
